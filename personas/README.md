# Personas

This directory is the canonical source for persona startup prompts.

Each `*.md` file is intentionally provider-agnostic and concise so the same content can be passed to Codex, Claude, or Gemini as the initial prompt when a session starts.

Current personas:
- `brainstormer.md`
- `refactor.md`
- `tester.md`
- `reviewer.md`
- `slot-machine-bandit.md`
- `documenter.md`

These prompts are starting points. They should evolve as we observe real sessions and refine the behavior.
